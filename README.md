# Hangman0

English Description:
-------------------
Hangman is a classic word guessing game.
The player tries to guess a hidden word by
suggesting letters one at a time. For each
incorrect guess, a part of the hangman
figure is drawn. The game continues until
the word is completely guessed or the
hangman is fully drawn. It’s fun,
educational, and helps improve vocabulary
and spelling skills.

الوصف بالعربية:
-------------------
لعبة الرجل المشنوق هي لعبة كلاسيكية
لتخمين الكلمات. يحاول اللاعب معرفة
الكلمة المخفية من خلال اقتراح حروف
واحدة تلو الأخرى. مع كل تخمين خاطئ،
يتم رسم جزء من شكل الرجل المشنوق.
تستمر اللعبة حتى يتم تخمين الكلمة
بالكامل أو اكتمال رسم الرجل المشنوق.
إنها ممتعة وتعليمية، وتساعد على
تحسين المفردات ومهارات الإملاء.
